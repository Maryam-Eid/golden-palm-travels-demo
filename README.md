### Technical Stack

- **Backend:** PHP (Laravel)
- **Database:** MySQL
- **Architecture:** MVC, OOP principles
- **Features:** RESTful APIs, multi-language support
- **Authentication & Authorization:** Role-based access control
- **Admin Panel:**  Admin dashboard built with Laravel Filament
